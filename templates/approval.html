Required: When checked, this approver's approval is mandatory for the expense to be approved, regardless of other conditions.

When enabled, approvers must approve in the specified order (sequential). When disabled, all approvers can approve simultaneously (parallel).

%

Set the minimum percentage of approvers that must approve for the expense to be approved (applies only to non-required approvers).
